1. В базе данных Redis подберите коллекцию для подсчета посещений с определенных IP-адресов.

Вставим в коллекцию несколько значений
SADD ip '127.0.0.1'
SADD ip '127.0.0.2'
SADD ip '127.0.0.3'

Смотрим, какие уникальные значения появились у нас в коллекции
SMEMBERS ip

Считаем сумму значений
SCARD ip

2. При помощи базы данных Redis решите задачу поиска имени пользователя по электронному адресу и 
наоборот, поиск электронного адреса пользователя по его имени.

По ключу:

SET email ilya_alekseev@gmail.com
GET email

По по адресу:

SET ilya_alekseev@gmail.com email
GET ilya_alekseev@gmail.com
